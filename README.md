# Development Workflow

An idea-to-deployment workflow application with AI-assisted planning, task management, local Docker testing, and multi-cluster Kubernetes deployment via Skupper.

## Overview

This application guides you through the entire development lifecycle:

1. **Idea Capture** - Start with an idea and refine it collaboratively
2. **AI Planning** - Work with AI to create structured plans
3. **Task Management** - Break down plans into actionable tasks
4. **Local Testing** - Deploy and test in Docker containers
5. **Production Deploy** - Deploy to active-active Kubernetes clusters

## Architecture

- **Frontend:** Next.js 14+ with Tailwind CSS
- **Backend:** Bun + Hono
- **Database:** PostgreSQL (SQLite for development)
- **Infrastructure:** Docker + Kubernetes + Skupper
